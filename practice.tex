Quiz Practice \newline
Calculate probabilites and expectation values. \newline

$
\psi_n(x) = \sqrt{\frac{2}{L}}
\sin\left(\frac{n\pi x}{L}\right)
$ \newline

Find probability $n=2$,
from $x=\frac{L}{2}$ to $x=\frac{L}{4}$ \newline
$
\int \left|\psi(x)\right|^2 dx
$ \newline
$
\int \left|
    \sqrt{\frac{2}{L}}
    \sin\left(\frac{n\pi x}{L}\right)
  \right|^2 dx
$ \newline
$
\frac{2}{L}\int 
  \sin^2\left(\frac{n\pi x}{L}\right) dx \text{ ; }
u = \frac{n\pi x}{L} \text{ ; }
dx = \frac{du}{\frac{n\pi}{L}}
  = \frac{Ldu}{n\pi}
$ \newline
